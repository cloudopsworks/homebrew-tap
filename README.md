# Cloudopsworks Tap

## How do I install these formulae?

`brew install cloudopsworks/tap/<formula>`

Or `brew tap cloudopsworks/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cloudopsworks/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
